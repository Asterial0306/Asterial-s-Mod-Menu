# Asterial's Mod Menu

A mod menu for Minecraft 1.21.1 Fabric that lets you view, search, filter, and manage your installed mods.

## Features

- **Mod List** - Browse all installed mods with search, sort, and filter
- **Mod Details** - View ID, author, license, environment, and type for each mod
- **Quick Config** - Open mod configurations directly from the detail page
- **Homepage Link** - Safely visit mod homepages with a confirmation dialog
- **Bookmark** - Star your favorite mods and filter to show only bookmarks
- **Search History** - Recent searches appear when you click the search box
- **Smart Suggestions** - Auto-complete helps you find mods faster
- **M Key Shortcut** - Quickly open the mod menu from the main menu or pause screen
- **Conflict Detection** - Warns if other mod menu mods are installed
- **43 Languages** - Full translation support including English, Chinese, Japanese, Korean, and more

## Configuration

- Default sort order
- Remember filters and sort state
- Row spacing (compact / normal / spacious)
- Show / hide dependencies
- Hide Fabric API sub-modules
- Show author information
- Search suggestions
- Search history
- Bookmark feature
- M key shortcut

## Requirements

- Minecraft 1.21.1
- Fabric Loader 0.19.2+
- Fabric API
- Java 21+
